THE COLLECTION

Search thousands of artworks in the museum’s collection—from our renowned icons to lesser-known works from every corner of the globe—as well as our books, writings, reference materials, and other resources. Or you can create original artwork using the built-in image generator (limit 1 generation per minute). The image generator uses DALL·E, a 12-billion parameter neural network trained to generate images from text descriptions, using a dataset of text–image pairs. DALL-E has a diverse set of capabilities, including creating anthropomorphized versions of animals and objects, combining unrelated concepts in plausible ways, rendering text, and applying transformations to existing images.
